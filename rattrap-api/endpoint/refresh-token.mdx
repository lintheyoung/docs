---
title: '刷新 Token'
openapi: 'POST /auth/v1/token'
---

<RequestExample>
```bash cURL
curl -X POST \
  'https://lrsppxarhxvxcflougbp.supabase.co/auth/v1/token?grant_type=refresh_token' \
  -H "apikey: your-supabase-anon-key" \
  -H "Content-Type: application/json" \
  -d '{
    "refresh_token": "v1.MRjefG7..."
  }'
```

```javascript Node.js
const res = await fetch("https://lrsppxarhxvxcflougbp.supabase.co/auth/v1/token?grant_type=refresh_token", {
  method: "POST",
  headers: {
    "apikey": "your-supabase-anon-key",
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    refresh_token: "v1.MRjefG7..."
  })
});

const { access_token, refresh_token } = await res.json();
console.log("Token 已刷新");
```
</RequestExample>

<ResponseExample>

</ResponseExample>

## 使用说明

当 access_token 过期时，使用此接口获取新的 Token，无需重新登录。

<Note>
替换 `your-supabase-anon-key` 为你的 Supabase 项目的 anon key。
</Note>

<Warning>
- 所有 Supabase API 调用都需要 `apikey` 请求头
- refresh_token 也有有效期，如果 refresh_token 过期，需要重新登录
</Warning>
